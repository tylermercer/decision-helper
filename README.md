# Decision Helper

A small web app for helping the user make difficult decisions

## Building

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running on localhost

```sh
npm run start:dev
```
